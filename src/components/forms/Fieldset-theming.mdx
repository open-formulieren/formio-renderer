# Fieldset theming

⚠️ The `Fieldset` component is currently not based on NL Design System (community) components,
though we plan to address this in a future version.

It does (reuse) some design tokens from the following components:

- `utrecht-form-field`
- `utrecht-form-fieldset`
- `utrecht-alert`
