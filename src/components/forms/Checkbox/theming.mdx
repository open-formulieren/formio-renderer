# Checkbox theming

The `Checkbox` form field component is implemented on top of some Utrecht community components:

- `utrecht-custom-checkbox`
  [design tokens](https://nl-design-system.github.io/utrecht/storybook/?path=/story/css_css-checkbox--design-tokens)
- `utrecht-form-field`
  [design tokens](https://nl-design-system.github.io/utrecht/storybook/?path=/story/css_css-form-field--design-tokens)

## Open Forms overrides

### Extensions

Open Forms extends the `utrecht-checkbox` component, targetting the `utrecht-form-field--openforms`
modifier, using a number of custom design tokens. These are available to all themes.

### Theme-specific overrides

The default Open Forms theme overrides some checkbox styles. You don't get these overrides unless
you use the `.openforms-theme` or implement similar CSS-level overrides.

- Layout arrangement.
- Focus outline management to handle the altered layout arrangement.

All the styles are scoped to Open Forms extensions on elements and should not affect elements
rendered by other means than Open Forms.
