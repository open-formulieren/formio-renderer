# TextField

A Text Field can be used for short and general text input. There are options to define input masks
and validations, allowing users to mold information into desired formats.

**Controlled TextField** `<TextField value={value} callbacks={{onChange: onChange}}/>`

<Canvas>
  <Story
    name="Controlled TextField"
    args={{
      component: {
        type: 'textfield',
        label: 'first name',
        placeholder: 'Enter your first name.',
        description: 'Enter your first name',
      },
      value: 'The quick brown fox jumps over the lazy dog.',
    }}
    decorators={[
      (Story, args) => {
        const [state, setState] = useState(args.args.value);
        args.args.value = state;
        args.args.callbacks = {onChange: e => setState(e.target.value)};
        return Story();
      },
    ]}
  >
    {Template.bind({})}
  </Story>
</Canvas>
